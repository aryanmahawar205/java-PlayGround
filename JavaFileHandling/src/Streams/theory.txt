Stream is basically a sequence of data
This data can be of two types - byte or unicode characters
Stream is just an Abstraction that Java provides
Java implements these within class hierarchy in java.io package

There are two types of Streams in Java - Byte Stream & Character Stream

The Byte Streams are defined in two classes - InputStream and OutputStream (both these classes extend the Object class)
The Character Streams are defined in two classes - Reader and Writer.
